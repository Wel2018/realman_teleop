# 使用说明

## 版本对应表

| 发布时间            | 基础库版本                                        | 程序版本                                  |
| ------------------- | ------------------------------------------------- | ----------------------------------------- |
| 2026-01-14 14:55:36 | `robotic_toolbox-3.2.7.post20260114-py3-none-any` | `dist_realman_teleop_2026-01-14-14.41.50` |

## 升级基础库

```bash
./realman_teleop_env/Scripts/pip3.exe install ./robotic_toolbox-3.2.7.post20260114-py3-none-any.whl
```

## 安装驱动 *

右键管理员运行 `prepare_realman_teleop_app.bat` 

作用：

- SpaceMouse 驱动
- 开机自启动配置

## 手动启动程序

右键管理员运行 `start_realman_teleop_app.bat`

## 创建环境 *

```sh
conda create -n dist python==3.11
conda activate dist
pip install ./resources/robotic_toolbox.whl
pip install -r .resources/requirements.txt
```
